# Forkify

A recipe search engine to find, collect, and create recipes.

## Features
- Search for recipes from a large database.
- Adjust serving sizes.
- Save recipes.
- Create recipes without logging in.
- Modern and clean UI.

## Built With
- HTML
- CSS
- JavaScript

- JavaScript
- DOM manipulation
- REST API
- Simple Database
- GIT
